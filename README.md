# eta.bar
Sample project using Angular2 with webpack

Set index.html values with
* config/webpack.common.js
* config/head-config.commons.js

# Start up for coding
```
npm run watch:prod
```
Leaves a copy of the packed version in backend's public folder, then
```
cd backend
node app.js
```
